## 简介

MitOS 是一个简单的 32 位操作系统，其大小可容纳在一张 1.44M 软盘下，支持图形用户界面、鼠标操作、内存管理、分段、多任务等内容。系统采用汇编语言和 C 语言编写，运行速度迅捷。

## 运行环境

要安装或运行 MitOS ，您的计算机需满足以下条件：
  * Intel 486 处理器或更高
  * 32 MiB 可用 RAM
  * 3.5 inch 软盘或 CD-ROM 驱动器

您也可以在虚拟机上体验 MitOS 。
